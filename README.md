# vue-calender

## vue实现小日历组件

### 参数
  * ``markList`` 需要标记的日期 时间字符串数组格式 yyyy-M-d，如2021-9-8(不是2021-09-08)
  * ``colorGroup`` 标记日期是否分组启用时 markList 格式为[{date: yyyy-M-d, group: xxxx}]，同一组的 group 写成一样
